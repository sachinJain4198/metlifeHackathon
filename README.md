# MetLife Hackathon React App

React + Vite app using Redux Toolkit, React Router, MUI, Bootstrap, and Axios.

## Run

```powershell
npm install
npm run dev -- --host
```

## Scripts

- dev: start Vite dev server
- build: production build
- preview: preview build

## Structure

- src/app: Redux store
- src/features: Redux slices
- src/pages: route pages
- src/components: reusable UI (e.g., Layout)
* Routing is defined in `src/App.jsx`
- src/services: axios api and mock data
- src/theme: MUI theme
